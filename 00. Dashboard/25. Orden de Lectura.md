# 🛤️ Orden de Lectura (Biblioteca Mínima)

Una cosa es tener 50 obras maestras y otra saber en qué orden atacarlas para no frustrarse.
Este es el progreso secuencial sugerido de los 50 libros de la Biblioteca Mínima.

## 🟢 Etapa 1: Los Cimientos (Hábitos, Ética y Lógica Base)
*No empezamos con Kant ni con Papers de IA. Empezamos ajustando el sistema humano.*
1. **Atomic Habits** (Clear) -> Para instalar el hábito de lectura y estudio.
2. **Apología de Sócrates** (Platón) -> Para activar el cuestionamiento.
3. **Meditaciones** (Marco Aurelio) -> Para forjar autocontrol temprano.
4. **The Pragmatic Programmer** (Thomas/Hunt) -> Para adoptar la mentalidad de ingeniero.
5. **Basic Economics** (Sowell) -> Para entender incentivos básicos en la vida y mercado.
6. **Sapiens** (Harari) -> Para ganar la visión macro de la civilización.

## 🟡 Etapa 2: Estructuras y Mecanismos (Sistemas, Mente y Negociación)
*Aquí empezamos a desarmar cómo funciona el mundo real.*
7. **Thinking, Fast and Slow** (Kahneman) -> La mente tiene fallos sistemáticos.
8. **Clean Code** (Martin) -> Profesionalismo en la ejecución.
9. **Never Split the Difference** (Voss) -> Aplicación táctica de la psicología.
10. **El Hombre en busca de Sentido** (Frankl) -> Resiliencia humana profunda.
11. **Historia de Roma** (Montanelli) -> Los ciclos del poder y el estado.
12. **El Arte de la Guerra** (Sun Tzu) -> Gestión de recursos y conflictos.
13. **Code: The Hidden Language** (Petzold) -> Bajando al silicio.

## 🟠 Etapa 3: Sistemas Complejos y Estrategia
*Acá las cosas se ponen técnicas e interdisciplinarias.*
14. **Clean Architecture** (Martin) -> Pensamiento abstracto a nivel macro.
15. **El Príncipe** (Maquiavelo) -> Realpolitik. Cero idealismos.
16. **Freakonomics** (Levitt/Dubner) -> Correlación vs Causalidad.
17. **Steve Jobs** (Isaacson) -> Cruzando tecnología con humanidades.
18. **1984** (Orwell) -> El lenguaje como control de sistemas políticos.
19. **On Writing Well** (Zinsser) -> Aprender a pensar de forma estructurada en papel.
20. **Good Strategy / Bad Strategy** (Rumelt) -> Planificación a largo plazo.

## 🔴 Etapa 4: Disrupción, Escala y Abstracción Superior
*Libros de alta dificultad o impacto conceptual.*
21. **Designing Data-Intensive Applications** (Kleppmann) -> Escala técnica brutal.
22. **The Black Swan** (Taleb) -> El impacto de lo altamente improbable en economía/historia.
23. **El Gen Egoísta** (Dawkins) -> Algoritmos biológicos y evolución de sistemas.
24. **Zero to One** (Thiel) -> Filosofía empresarial contrariana.
25. **A Philosophy of Software Design** (Ousterhout) -> Reducción de entropía (Complejidad).
26. **The Innovator's Dilemma** (Christensen) -> Ciclos de muerte tecnológica.
27. **Leonardo da Vinci** (Isaacson) -> La interdisciplinariedad máxima.

*(El resto de los 50 libros se ubicarán en los bloques siguientes a medida que superemos la Etapa 4).*
