# 📖 Glosario Personal

Conceptos clave explicados con mis propias palabras de manera muy simple.

- **Zettelkasten:** Sistema para conectar ideas y crear notas atómicas.
- **Second Brain:** Sistema externo (como este Vault) donde descargo información para liberar espacio mental.
- **Interdisciplinariedad:** Cruzar fronteras entre diferentes ciencias y humanidades.
