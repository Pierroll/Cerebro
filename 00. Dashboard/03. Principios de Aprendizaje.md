# 🧠 Principios de Aprendizaje

Reglas inquebrantables para mi estudio:

1. **Comprensión sobre memorización:** Si no puedo explicarlo simple, no lo entiendo.
2. **Conexión:** Relacionar conceptos de diferentes áreas (ej. Ingeniería + Psicología).
3. **Mis propias palabras:** Nunca hacer copiar y pegar (Copy-Paste) de texto.
4. **Práctica:** Aplicar el conocimiento a problemas reales de mi entorno.
